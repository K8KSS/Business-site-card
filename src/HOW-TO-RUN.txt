========================================
  HOW TO RUN THE PROJECT
========================================

OPTION 1 (FASTEST):
  Double-click: RUN.bat

OPTION 2 (If RUN.bat doesn't work):
  Double-click: FIX-AND-RUN.bat
  (This will fix npm cache and run the project)

OPTION 3 (Terminal):
  Open PowerShell or CMD in this folder
  Type: npx vite
  Press Enter

OPTION 4 (Super quick):
  Double-click: QUICK-RUN.bat

========================================

After running, open:
  http://localhost:5173

Admin panel:
  Press: Ctrl + Shift + A
  Password: admin

========================================

IMPORTANT:
- Backend is on Supabase (cloud)
- No local server needed
- No PostgreSQL needed
- Just run the frontend
