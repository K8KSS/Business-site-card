[ ] Test deployed website


TROUBLESHOOTING
===============

Problem: "npm command not found"
---------------------------------
[ ] Install Node.js from: https://nodejs.org/
[ ] Restart terminal after installation


Problem: "Supabase CLI not installed"
--------------------------------------
[ ] Run: npm install -g supabase
[ ] OR use scoop: scoop install supabase
[ ] Restart terminal


Problem: "HTTP 404 errors in admin"
------------------------------------
[ ] Check that Edge Functions are deployed
[ ] Run: DEPLOY_EDGE_FUNCTIONS.bat
[ ] Check deployment URL works:
    https://oqazxfewxttctehgftuy.supabase.co/functions/v1/make-server-322de762/health


Problem: "Styles not working"
------------------------------
[ ] Hard refresh browser: Ctrl + Shift + R
[ ] Clear Vite cache: delete node_modules/.vite/
[ ] Restart dev server


Problem: "Port 5173 already in use"
------------------------------------
[ ] Stop other processes using port
[ ] OR change port in vite.config.ts
