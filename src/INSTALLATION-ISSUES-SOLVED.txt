============================================================
  WHY "vite is not recognized" ERROR HAPPENS
============================================================

Possible reasons:

1. You're in the wrong folder
   → Make sure package.json is in current folder

2. node_modules were never installed
   → Run REINSTALL.bat

3. Installation was interrupted
   → Run REINSTALL.bat

4. Files were copied without node_modules
   → Run REINSTALL.bat

5. Different project folder opened
   → Navigate to correct folder, run REINSTALL.bat


============================================================
  HOW TO CHECK IF YOU'RE IN RIGHT FOLDER
============================================================

Run this command:
  dir

You should see:
  ✅ package.json
  ✅ App.tsx
  ✅ server (folder)
  ✅ components (folder)
  ✅ vite.config.ts
  ✅ node_modules (folder) <- after installation

If you DON'T see these files, you're in wrong folder!


============================================================
  WHAT REINSTALL.BAT DOES
============================================================

1. Checks package.json exists
2. Stops running Node processes
3. Deletes node_modules folder
4. Deletes package-lock.json
5. Clears npm cache
6. Installs dependencies with --legacy-peer-deps
7. If fails, tries with --force
8. Creates database if doesn't exist
9. Ready to use!


============================================================
  MANUAL INSTALLATION (IF BAT FILES DON'T WORK)
============================================================

Open CMD in project folder:

1. Click address bar in File Explorer
2. Type: cmd
3. Press Enter

Then run:

npm cache clean --force
rmdir /s /q node_modules
del package-lock.json
npm install --legacy-peer-deps
npm run setup
npm run dev
