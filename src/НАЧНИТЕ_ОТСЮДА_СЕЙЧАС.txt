Вариант 1: Только локальный запуск (демо-режим)
────────────────────────────────────────────────

1. Дважды кликните: START.bat
2. Выберите пункт 2 (Run development server)
3. Сайт откроется автоматически в браузере
4. ✅ Готово! Можете тестировать сайт

Примечание: Админ-панель будет работать с демо-данными в памяти.
После перезагрузки страницы изменения сбросятся.


Вариант 2: Полная настройка с Supabase (рекомендуется)
───────────────────────────────────────────────────────

Шаг 1: Установите Supabase CLI
   scoop install supabase
   (или: npm install -g supabase)

Шаг 2: Разверните Edge Functions
   Дважды кликните: DEPLOY_EDGE_FUNCTIONS.bat

Шаг 3: Запустите сайт
   Дважды кликните: START.bat

Шаг 4: Проверьте работу
   - Откройте админ-панель: Ctrl + Shift + A
   - Загрузите тестовое фото
   - Все должно работать без ошибок 404!

❓ Нужен ли мне аккаунт Supabase?
   → Для демо-режима НЕТ
   → Для сохранения данных ДА (бесплатный план подходит)

❓ Как открыть админ-панель?
   → Нажмите Ctrl + Shift + A
   → Или тройной клик по логотипу
   → Пароль по умолчанию: admin

❓ Получаю ошибку "Docker is not running"
   → Это НЕ ошибка! Docker не нужен
   → Edge Function развертывается на облаке Supabase
   → Можете спокойно игнорировать это предупреждение

❓ После деплоя получаю 404
   → Подождите 1-2 минуты
   → Проверьте URL: должен содержать /make-server-322de762/
   → Откройте логи в Supabase Dashboard

❓ Где посмотреть логи?
   → https://supabase.com/dashboard/project/oqazxfewxttctehgftuy/logs/edge-functions

❓ Как изменить пароль админ-панели?
   → В Supabase Dashboard → Project Settings → Edge Functions
   → Добавьте переменную окружения ADMIN_PASSWORD

❓ Можно ли использовать без интернета?
   → ДА, в демо-режиме (без Supabase)
   → НЕТ, для сохранения данных нужен интернет
