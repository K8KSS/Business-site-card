============================================================
                    ESSENTIAL COMMANDS
         Music Teacher Website - Command Reference
============================================================


------------------------------------------------------------
  INSTALLATION (First Time Only)
------------------------------------------------------------

Method 1 - Automatic (Recommended):

    Double-click: INSTALL-NOW.bat


Method 2 - Manual:

    npm install
    npm run setup


------------------------------------------------------------
  STARTING THE WEBSITE (Every Time)
------------------------------------------------------------

Method 1 - Automatic:

    Double-click: start-windows.bat


Method 2 - Manual:

    npm run dev


------------------------------------------------------------
  OPENING IN BROWSER
------------------------------------------------------------

    http://localhost:5173


------------------------------------------------------------
  ADMIN PANEL ACCESS
------------------------------------------------------------

    1. Click logo 3 times
    2. Password: admin


------------------------------------------------------------
  STOPPING THE SERVER
------------------------------------------------------------

    Press: Ctrl + C
    Then: Y (confirm)


------------------------------------------------------------
  TROUBLESHOOTING
------------------------------------------------------------

If you get errors, run:

    INSTALL-NOW.bat

Or for diagnostics:

    diagnose.bat


------------------------------------------------------------
  AVAILABLE SCRIPTS
------------------------------------------------------------

npm install          Install dependencies
npm run setup        Create database
npm run dev          Start development server
npm run build        Build for production
npm run preview      Preview production build
npm run check        Check system requirements


------------------------------------------------------------
  USEFUL FILES
------------------------------------------------------------

INSTALL-NOW.bat      Automatic installation
start-windows.bat    Start website
diagnose.bat         System diagnostics

READ-ME-FIRST.txt    Quick overview
QUICK-START.txt      Installation guide
QUICK-FIX.md         Problem solving
USER_GUIDE.md        How to use


------------------------------------------------------------
  FOLDER STRUCTURE
------------------------------------------------------------

/server              Backend (Express + SQLite)
/components          React components
/styles              CSS styles
/public              Static assets

/server/database.sqlite    Your data
/server/uploads            Uploaded files


------------------------------------------------------------
  BACKUP
------------------------------------------------------------

To backup your data, copy:

    /server/database.sqlite
    /server/uploads/


------------------------------------------------------------
  COMMON ISSUES
------------------------------------------------------------

"Missing script: setup"
    -> Run: INSTALL-NOW.bat

"Port already in use"
    -> Stop other servers or change port

"Node.js not found"
    -> Install from: https://nodejs.org/

"npm not found"
    -> Restart after Node.js installation

  PORTS USED
