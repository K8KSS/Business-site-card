============================================================
  TAILWIND 4.0 VS 3.X
============================================================

TAILWIND 3.X (old):
  @tailwind base;
  @tailwind components;
  @tailwind utilities;

TAILWIND 4.0 (new):
  @import "tailwindcss";

We're using Tailwind 4.0, so we need the @import syntax!


============================================================
  TECHNICAL DETAILS
============================================================

How Tailwind CSS works:

1. You write: class="bg-blue-500 text-white p-4"

2. Tailwind generates CSS:
   .bg-blue-500 { background-color: #3b82f6; }
   .text-white { color: white; }
   .p-4 { padding: 1rem; }

3. Browser applies styles to elements


Without @import "tailwindcss":
  - Step 2 DOESN'T HAPPEN
  - No CSS generated
  - Classes do nothing
  - Plain HTML only


With @import "tailwindcss":
  - Step 2 WORKS
  - CSS generated
  - Styles applied
  - Beautiful website!

When creating new Tailwind CSS projects:

ALWAYS add to your main CSS file:
  @import "tailwindcss";

This is the FIRST line in globals.css!


For Tailwind 4.0 projects:
  1. Install: npm install tailwindcss@4.0.0
  2. Create CSS file
  3. Add: @import "tailwindcss";
  4. Import CSS in main.tsx
  5. Use Tailwind classes
  6. Done!
