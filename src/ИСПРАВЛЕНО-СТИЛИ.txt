============================================================
  TAILWIND 4.0 VS 3.X
============================================================

TAILWIND 3.X (старый):
  @tailwind base;
  @tailwind components;
  @tailwind utilities;

TAILWIND 4.0 (новый):
  @import "tailwindcss";

Мы используем Tailwind 4.0, поэтому нужен синтаксис @import!


============================================================
  ТЕХНИЧЕСКИЕ ДЕТАЛИ
============================================================

Как работает Tailwind CSS:

1. Вы пишете: class="bg-blue-500 text-white p-4"

2. Tailwind генерирует CSS:
   .bg-blue-500 { background-color: #3b82f6; }
   .text-white { color: white; }
   .p-4 { padding: 1rem; }

3. Браузер применяет стили к элементам


Без @import "tailwindcss":
  - Шаг 2 НЕ ПРОИСХОДИТ
  - CSS не генерируется
  - Классы ничего не делают
  - Только простой HTML


С @import "tailwindcss":
  - Шаг 2 РАБОТАЕТ
  - CSS генерируется
  - Стили применяются
  - Красивый сайт!

============================================================
  НА БУДУЩЕЕ:
============================================================

При создании новых проектов с Tailwind CSS:

ВСЕГДА добавляйте в главный CSS файл:
  @import "tailwindcss";

Это ПЕРВАЯ строка в globals.css!


Для проектов с Tailwind 4.0:
  1. Установить: npm install tailwindcss@4.0.0
  2. Создать CSS файл
  3. Добавить: @import "tailwindcss";
  4. Импортировать CSS в main.tsx
  5. Использовать классы Tailwind
  6. Готово!
