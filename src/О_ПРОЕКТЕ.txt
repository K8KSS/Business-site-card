Профессиональный сайт-портфолио для музыкального руководителя 
детского сада с полным функционалом и современным дизайном.

┌──────────────────────────────────────────────────────────────────┐
│  ОСНОВНЫЕ РАЗДЕЛЫ САЙТА                                          │
└──────────────────────────────────────────────────────────────────┘

✓ Главная - Приветственная страница с анимацией
✓ О себе - Персональная информация, образование, опыт
✓ Публикации - 10 категорий методических материалов
✓ Фотоальбомы - Коллекции фотографий с мероприятий
✓ Достижения - Витрина наград и сертификатов
✓ Портфолио - Дипломы и благодарности
✓ Отзывы - Отзывы родителей и коллег с модерацией
✓ Связь - Форма обратной связи
✓ Аудио - Музыкальная библиотека с плеером
✓ Видео - Видеогалерея с интеграцией VK

Скрытая админ-панель для управления контентом:

🔐 Вход в админку:
   • Ctrl + Shift + A
   • Тройной клик по логотипу

📝 Возможности:
   • Добавление/редактирование публикаций
   • Управление фотоальбомами
   • Загрузка аудио и видео
   • Модерация отзывов
   • Управление портфолио
   • Просмотр сообщений

┌──────────────────────────────────────────────────────────────────┐
│  ТЕХНОЛОГИИ                                                      │
└──────────────────────────────────────────────────────────────────┘

Frontend:
   • React 18.3 - UI библиотека
   • TypeScript - Типизация
   • Vite 6 - Сборщик и dev-сервер
   • Tailwind CSS v4 - Стили
   • Motion (Framer Motion) - Анимации
   • Lucide React - Иконки

Backend & БД:
   • Supabase - Backend-as-a-Service
   • PostgreSQL - База данных
   • Supabase Storage - Хранение файлов

UI компоненты:
   • Shadcn/ui - Набор готовых компонентов
   • Radix UI - Accessibility-first примитивы

Windows:          Запустите START.bat

Терминал:
   npm install       - Установка зависимостей
   npm run dev:client - Запуск разработки
   npm run build     - Сборка проекта
   npm run preview   - Просмотр сборки
   npm run start     - Сборка + просмотр
