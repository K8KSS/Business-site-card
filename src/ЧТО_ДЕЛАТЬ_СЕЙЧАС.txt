❌ ЕСЛИ СТИЛЕЙ НЕТ (белый фон, черный текст):

  1. Остановите сервер (Ctrl + C)
  
  2. Выполните команды:
     rmdir /s /q node_modules
     rmdir /s /q node_modules\.vite  
     del package-lock.json
     npm install
  
  3. Запустите снова:
     npm run dev:client
  
  4. В браузере:
     Ctrl + Shift + R (жесткая перезагрузка)
