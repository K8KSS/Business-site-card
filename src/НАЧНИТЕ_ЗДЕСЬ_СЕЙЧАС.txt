🚀 ВАРИАНТ 1: АВТОМАТИЧЕСКИЙ ЗАПУСК (РЕКОМЕНДУЕТСЯ)
───────────────────────────────────────────────────

Просто дважды кликните по одному из файлов:

  1️⃣ START.bat
     └─→ Главное меню с выбором действий
         Выберите пункт 2 (Run development server)

  2️⃣ DEPLOY_EDGE_FUNCTIONS.bat
     └─→ Автоматический деплой Edge Functions
         Исправит все HTTP 404 ошибки


🔧 ВАРИАНТ 2: РУЧНОЙ ЗАПУСК
────────────────────────────

Откройте терминал и выполните:

  1. Установите зависимости:
     npm install

  2. Запустите сайт:
     npm run dev:client

  3. Откройте браузер:
     http://localhost:5173
