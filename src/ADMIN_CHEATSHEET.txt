└─────────────────────────────────────────────────────────────┘

  1. Откройте файл:  utils/supabase/info.tsx
  
  2. Замените значения:
     export const projectId = 'ваш-project-id';
     export const publicAnonKey = 'ваш-anon-ключ';

  3. Найти ключи в Supabase:
     Dashboard → Settings → API
     • Project URL → берём ID из URL
     • anon public → копируем ключ

    npm install -g supabase

    .\DEPLOY_EDGE_FUNCTIONS.bat

┌─────────────────────────────────────────────────────────────┐
│  🌐 ДЕПЛОЙ НА NETLIFY                                        │
└─────────────────────────────────────────────────────────────┘

  Вариант 1 - Ручной (быстрый):
    1. npm run build
    2. Зайти на netlify.com
    3. Перетащить папку dist
    
  Вариант 2 - Автоматический (через Git):
    1. Загрузить проект на GitHub
    2. В Netlify: Import from Git
    3. Build command: npm run build
    4. Publish directory: dist
    5. Environment variables:
       VITE_SUPABASE_URL
       VITE_SUPABASE_ANON_KEY

│  🎬 КАК ДОБАВИТЬ ВИДЕО ИЗ VK                                 │
└─────────────────────────────────────────────────────────────┘

  1. Откройте видео на VK
  2. Нажмите "Поделиться" → "HTML-код"
  3. Скопируйте URL из src="..."
  
  Пример правильной ссылки:
  https://vk.com/video_ext.php?oid=-XXXXXXX&id=XXXXXXX
  
  ❌ Не используйте обычную ссылку типа:
     https://vk.com/video-XXXXXXX_XXXXXXX

  → Убедитесь что Edge Functions развернуты
  → Проверьте health endpoint
  → Посмотрите консоль на ошибки

  → Проверьте подключение к Supabase
  → В демо-режиме используются встроенные данные
  → Откройте консоль (F12) для диагностики
  
  Видео не воспроизводятся:
  → Используйте только embed ссылки VK
  → Проверьте что видео публичное
  
  Ошибка HTTP 404:
  → Прочитайте: HTTP_404_PROBLEM_SOLVED.md
  → Запустите: DEPLOY_EDGE_FUNCTIONS.bat
  
  Docker warning:
  → Это не ошибка! Docker не нужен для Supabase Cloud
  → Можно игнорировать это предупреждение
