============================================================
  COMMON QUESTIONS
============================================================

  Q: Which installer should I use?
  A: install-alternative.bat (safest)
  
  Q: What if both installers fail?
  A: Run diagnose.bat to check system
  
  Q: Is --legacy-peer-deps safe?
  A: Yes, for development. It just ignores
     strict version checking.
  
  Q: Will my data be lost?
  A: No, database is in server/database.sqlite
     It's only created if it doesn't exist.
  
  Q: Do I need to reinstall every time?
  A: No! Only once. Then use start-windows.bat
